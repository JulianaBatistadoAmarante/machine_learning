# Dataset Kit 01 — Alunos

Este kit simula uma base escolar utilizada para ensinar exploração, limpeza, preparação de dados e Machine Learning.

## Problema

A escola deseja prever se um aluno será aprovado com base em idade, curso, nota, frequência e atividades entregues.

## Arquivos CSV

- `csv/original/alunos_original.csv`: dados consistentes.
- `csv/sujo/alunos_sujo.csv`: dados com problemas comuns.
- `csv/muito_sujo/alunos_muito_sujo.csv`: dados com problemas avançados.
- `csv/tratado/alunos_tratado.csv`: versão corrigida.

## Tarefa principal

Preparar os dados para treinar um modelo de classificação da coluna `Aprovado`.
