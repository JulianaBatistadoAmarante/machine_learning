# Lista de Exercícios — Alunos

## Parte 1 — Exploração

1. Abra o arquivo `alunos_sujo.csv`.
2. Exiba as cinco primeiras linhas.
3. Exiba cinco registros aleatórios.
4. Mostre o número de linhas e colunas.
5. Liste os tipos de dados de cada coluna.

## Parte 2 — Problemas nos dados

6. Verifique valores nulos.
7. Identifique registros duplicados.
8. Liste os valores únicos da coluna `Curso`.
9. Verifique se existem idades inválidas.
10. Verifique se existem notas acima de 10.
11. Verifique se existem frequências acima de 100.

## Parte 3 — Limpeza

12. Remova duplicados.
13. Remova espaços extras dos nomes.
14. Padronize os cursos.
15. Preencha notas nulas com a média.
16. Corrija frequências acima de 100.
17. Corrija idades negativas.
18. Converta a coluna de data para datetime.

## Parte 4 — Preparação para Machine Learning

19. Crie `X` com as features numéricas.
20. Crie `y` com o target `Aprovado`.
21. Salve o arquivo tratado como `alunos_tratado.csv`.

## Desafio

Repita o processo com `alunos_muito_sujo.csv`, registrando todas as decisões tomadas.
