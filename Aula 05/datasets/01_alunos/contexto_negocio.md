# Contexto de Negócio

Uma escola deseja construir um sistema simples de apoio pedagógico para identificar alunos com maior chance de aprovação ou reprovação.

Antes de treinar um modelo, a equipe recebeu uma base de dados extraída do sistema acadêmico. O problema é que os dados possuem inconsistências comuns em sistemas reais.

## Objetivo do projeto

Preparar o dataset para que ele possa ser utilizado em um modelo de Machine Learning.

## Pergunta principal

Com base em idade, curso, nota, frequência e atividades entregues, é possível prever se o aluno será aprovado?

## Valor pedagógico

Este dataset permite ensinar:

- leitura de CSV;
- exploração com Pandas;
- valores nulos;
- duplicados;
- padronização textual;
- tratamento de valores inválidos;
- conversão de datas;
- preparação de `X` e `y`;
- classificação binária.
