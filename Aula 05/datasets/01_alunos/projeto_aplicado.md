# Projeto Aplicado — Alunos

## Situação-problema

A coordenação pedagógica deseja identificar alunos com maior risco de reprovação.

Você recebeu uma base de dados com problemas e deverá prepará-la para uso em Machine Learning.

## Entregáveis

1. Diagnóstico dos problemas encontrados.
2. Dataset limpo.
3. Código da limpeza.
4. Justificativa das decisões tomadas.
5. Separação de features e target.
6. Pequeno relatório final.

## Pergunta orientadora

Quais informações parecem mais relevantes para prever aprovação?

## Critérios de qualidade

- O arquivo final não deve conter duplicados.
- O arquivo final não deve conter valores nulos sem justificativa.
- Textos devem estar padronizados.
- Idade, nota e frequência devem respeitar os domínios definidos.
