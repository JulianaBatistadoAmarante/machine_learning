# Guia do Professor — Dataset Alunos

## Objetivos de aprendizagem

Ao final da atividade, o aluno deverá ser capaz de:

- abrir arquivos CSV com Pandas;
- explorar um DataFrame;
- identificar valores nulos;
- identificar duplicados;
- padronizar textos;
- corrigir valores inválidos;
- justificar decisões de limpeza;
- preparar dados para Machine Learning.

## Pré-requisitos

- Python básico;
- Pandas introdutório;
- conceitos de dataset, feature e target.

## Tempo sugerido

- Turma iniciante: 4 horas.
- Turma intermediária: 2 horas.
- Turma avançada: 1 hora com `alunos_muito_sujo.csv`.

## Dificuldades esperadas

- Confundir zero com valor nulo.
- Usar `dropna()` sem refletir.
- Esquecer de salvar o arquivo tratado.
- Não padronizar textos antes de contar categorias.
- Não justificar decisões.

## Estratégia didática

1. Peça que os alunos explorem antes de limpar.
2. Liste os problemas encontrados no quadro.
3. Discuta diferentes formas de tratamento.
4. Faça a limpeza em etapas.
5. Finalize com uma reflexão sobre qualidade dos dados.
