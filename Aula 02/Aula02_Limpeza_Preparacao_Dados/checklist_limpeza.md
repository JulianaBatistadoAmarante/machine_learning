# Checklist de Limpeza de Dados

**© @karizeviecelli - 2026**

Antes de considerar o dataset pronto:

- [ ] Fiz uma cópia do arquivo original.
- [ ] Verifiquei `df.shape`.
- [ ] Verifiquei `df.info()`.
- [ ] Verifiquei `df.describe()`.
- [ ] Contei os valores nulos.
- [ ] Analisei como preencher ou remover os nulos.
- [ ] Contei e analisei duplicados.
- [ ] Padronizei os textos.
- [ ] Corrigi os tipos das colunas.
- [ ] Procurei valores fora dos limites permitidos.
- [ ] Validei novamente o dataset.
- [ ] Salvei uma nova versão com `index=False`.
