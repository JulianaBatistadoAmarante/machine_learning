# Dataset 01 — Alunos

## Objetivo
Usado nas primeiras aulas para explorar dados, entender features/target e treinar o primeiro modelo de classificação.

## Problema
Prever se um aluno será aprovado.

## Target
`Aprovado`

## Principais colunas
- `Idade`
- `Curso`
- `Horas_Estudo`
- `Nota`
- `Frequencia`
- `Atividades_Entregues`
- `Aprovado`

## Arquivos
- `alunos.csv`: versão limpa
- `alunos_sujo.csv`: versão com nulos, duplicados, textos despadronizados e valores inválidos
